CSS侦动画、补间动画

关键侦动画和补间动画的区别

关键侦动画就是需要自己完成动画的关键帧，需要知道动画的关键侦（包括起始和末尾），而补间动画则是只知道起始和末尾状态，对中间过渡进行补间的。 关键侦动画可以实现无限循环、动画反转等效果，而补间动画只能实现一次性的过渡效果。

1. css侦动画

CSS3 Animation 由三部分组成

  • 动画属性(properties) - 定义动画的播放时长，播放次数，以及用何种时间函数去播放动画等
  • 关键帧(keyframes) - 定义动画在不同阶段的状态
  • css属性 - 就是css元素不同关键帧下的状态

举个🌰，实现一个简单的圆呼吸效果

🤔首先定义动画属性，依次是： > 动画名称、持续时间、时间函数、延迟时间、循环次数、是否轮流反向播放 ，animation: name duration timing-function delay iteration-count direction

⛽️接下来，定义keyframes

@keyframes 名称

😄最后，定义css变化属性

0%(from) 属性 属性值

50% 属性 属性值

100%(to) 属性 属性值

直接上代码

.circle {
  width: 50px;
  height: 50px;
  background: rgba(0, 0, 0, .5);
  animation: breath-circle 2s ease-out 1s infinite;
}
@keyframes breath-circle {
  0% { transform: scale(1); background: rgba(0, 0, 0, .5); }
  25% { transform: scale(.9); background: rgba(0, 0, 0, .4); }
  50% { transform: scale(.8); background: rgba(0, 0, 0, .3); }
  75% { transform: scale(.9); background: rgba(0, 0, 0, .4); }
  100% { transform: scale(1); background: rgba(0, 0, 0, .5); }
}

如果需要兼容浏览器，根据需要设置-webkit-这些前缀

2. css补间动画

补间动画我理解就是对transition属性的灵活运用

  • transition 属性 时间函数 渐变时长 延迟时长

举个🌰，实现一个从底部向上擦除的效果

代码撸起来 go go go…

<div class="box"></div>

.box{
  width: 100px;
  height: 100px;
  background: rgba(0, 0, 0, 0.7);
  pointer-events: none;
  mask: linear-gradient(to bottom, rgba(0, 0, 0, 1) 0%, rgba(0, 0, 0, 1) 100%) no-repeat center bottom;
  mask-size: 100% 0%;
  transition: mask-size ease-out 0.3s 0.3s;
}

// 在变化时添加box-go类，看看擦除效果
.box-go{
  pointer-events: auto;
  mask-size: 100% 100%;
}

其实大多数情况下都是渐变transform属性，因为transform属性的渐变不会触发回流、重绘，不吃性能，所以建议用transform属性做渐变动画
